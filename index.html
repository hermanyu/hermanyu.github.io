Hi! My name is Herman and I love building toy Machine Learning apps for everyone to play with. My dream is to build a "full stack machine learning" web application some day where people can do fun and silly things on the front-end, powered by ML and DL programs in the backend. I mainly code my ML projects in Python with PyTorch being my primary deep learning framework. I did start off using the Keras API for Tensorflow, but I switched to PyTorch out of curiosity and stayed because of preference. Currently, I am teaching myself how to build a full stack web application, starting with front-end development a la javascript. This personal page is being developed as the main project towards this endeavor.

I am a fervent believer in the concept of "open source knowledge"; science, technology, and mathematics should be enjoyed by everyone and not be sequestered away in the ivory towers of academia. As such, one of my primary life goals is to be a "Math and Tech communicator" where I can take arcane symbols in mathematics and technology textbooks and transform them into easy and interesting ideas for everyone to enjoy.

I was a PhD student in mathematics, where I was studying Tensor-Triangulated Categories and Tensor-Triangular Geometry. Around my 3rd year, I decided leave school and pursue a career in data science and machine learning. My passion for machine learning started in 2014 after seeing what Goodfellow et. al was able to do with GANs. Big data, deep learning, and advanced analytics have fascinated and mesmorized me ever since.

My first programming language is Python, which I picked up because of the readability and the amazing libraries for mathematics and machine learning. I wanted to expand my knowledge of software development, so I also taught myself some basic Java to develop an appreciation for Object-Oriented Programming. Since my dream is to build interactive web applications, I am currently trying to learn full stack development. This web page was started mainly in part to teach myself front-end web page building a la HTML, CSS, and Javascript.
